# `<PreviewText />`

- A React Native `<Text>` component that displays the preview color text.

| Property    |    Type     | Default | Description                            |
| :---------- | :---------: | :-----: | :------------------------------------- |
| colorFormat |  `string`   | `'hex'` | preview color text format: <formats /> |
| style       | `TextStyle` |    /    | preview text style                     |
