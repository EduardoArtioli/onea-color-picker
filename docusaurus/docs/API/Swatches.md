# `<Swatches />`

![swatches](../../../images/swatches.png)

- A list of colored swatches is used for quick color selection.

## Props

### `colors`

- Custom swatches colors.
- `type: string[]`
- `default: material colors`

### `style`

- Swatches container style.
- `type: ViewStyle`

### `swatchStyle`

- Swatch style.
- `type: ViewStyle`

:::info note

- Some style properties will be overwritten.

:::
